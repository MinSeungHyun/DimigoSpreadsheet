## 업데이트 예정

2. 가로로 돌렸을 때 또는 태블릿에서 나오는 뷰 설정
3. 야자 시작하기 전에 입력을 요구하는 알림 (또는 위치 확인)
4. 앱 처음 실행했을 때 팁 표시
5. 슬라이드 했다가 놓으면 부자연스러운 현상 수정
6. 로그인 화면에서 비밀번호 보기 버튼
7. 기타 사유 지우기
